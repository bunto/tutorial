--- layout: default title: Install Bunto with RubyGems number: 1 ---

The best way to install Bunto is via RubyGems. At the terminal prompt, simply run the following command to install Bunto:

$ gem install bunto -v 1.0.0

All of Bunto’s gem dependencies are automatically installed by the above command, so you won’t have to worry about them at all. If you have problems installing Bunto, report an issue so the Bunto community can improve the experience for everyone.
